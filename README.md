# 🌟 Overview

Traditional websites usually end with:

📞 Contact Form → Wait for Reply

This project takes a different approach.

Visitors interact with an **AI Sales Agent** that understands their business, recommends the right website solution, generates a personalized proposal, and automatically delivers it via email.

---

# ✨ Features

## 🤖 AI Consultation

* 💬 Natural conversation
* 🧠 Understands business requirements
* 🎯 Asks intelligent follow-up questions
* 💡 Suggests the most suitable website solution

---

## 📋 Requirement Collection

The AI gathers:

* 👤 Client Name
* 🏢 Business Name
* 💼 Business Type
* 🌐 Website Type
* 💰 Budget
* 📅 Timeline
* ⚙️ Required Features

---

## 📄 Proposal Generation

After understanding the client's needs, the AI automatically generates:

✅ Project Summary

✅ Recommended Website Type

✅ Suggested Features

✅ Estimated Budget

✅ Timeline

✅ Next Steps

---

## 📧 Automated Emails

Once the proposal is generated:

📩 Client receives a professional proposal.

📨 Abdul Web Agency receives the enquiry instantly.

No manual copy-pasting required.

# 🔒 Security

Security was considered from the beginning.

✅ Sensitive credentials are stored in a `.env` file.

This includes:

* 🔑 AI API Keys
* 📧 Email Credentials
* 🗄️ Supabase Keys
* ⚙️ Other Private Configuration

The `.env` file is excluded using `.gitignore`.

# 🚀 Installation

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file and configure:

* AI API Key
* Supabase Credentials
* Email Credentials

Run the project:

```bash
python app.py
```

✔️ Recommends website solutions

✔️ Generates professional proposals
